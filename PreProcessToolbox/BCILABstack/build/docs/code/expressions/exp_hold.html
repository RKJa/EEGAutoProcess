exp_hold

PURPOSE

Hold the given expression from being evaluated; can be released with exp_releasehold.

SYNOPSIS

function res = exp_hold(varargin)

DESCRIPTION 

 Hold the given expression from being evaluated; can be released with exp_releasehold.

 Out:
   Held-Expression ; The given expression(s), wrapped in a Hold expression
