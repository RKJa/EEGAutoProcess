gui_newapproach

PURPOSE

Open a dialog to select a new BCI approach

SYNOPSIS

function varargout = gui_newapproach(varargin)

DESCRIPTION 

 Open a dialog to select a new BCI approach
 Approach = gui_newapproch()

 Out:
   Approach : the newly created approach, or [] if the user cancelled the process
